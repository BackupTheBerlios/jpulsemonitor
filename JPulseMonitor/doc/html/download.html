Installation

The JPulseMonitor is written in Java (tm Sun Mircosystems). It runs on any platform that has a Java 2 Standard Edition Runtime Environment. If you encounter problems regarding the Java Version have a look at the FAQ. If you don't have Java installed, download and install the SDK or the JRE. Chances are quite high that you already have some version of JRE installed. So we just give it a try:

  1. Download the zip file and save it at a location of your choice. With some browsers, you need to right click (or click and hold) on the link and select 'Save file as...' from the context menu.
  2. Unpack it, leave all files in the same folder.
  3. On Windows (tm Microsoft) just doubleclick on the jPulesMonitor.jar. On any other system open a command shell and run java - jar jPulseMonitor.jar (in the folder where you unpacked the files in step 2).
  4. The application should pop up.

If nothing happens or you see some wired error showing up, you most probably do not have a Java installed. Go to Sun and get the newest JRE, install it and repeat step 3. Do also check the FAQ for known problems and bugs.
