I get a 'java.lang.VerifyError'?

I am terribly sorry about that. JPulseMonitor started as kind of a learning project I used to check out some new technologies. So I develop with the not yet stable Java 1.5, even though I compile JPulseMonitor with compatibility level 1.3, all JPulseMonitor versions above 0.1.4 seem to throw that error if run with an older Java engine. If I get a lot of requests about that error, I may step back to Java 1.4. Right now all you can do is either download a Java 1.5 JRE or use an older version of JPulseMonitor.

What libraries does it rely on?

  • The charts are generated with jfreechart
  • Older versions use JCalendar for date chooser components, now it is replaced by NachoCalendar
